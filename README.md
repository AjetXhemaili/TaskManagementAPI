# Task Management REST API

A secure, production-ready Backend API built with **ASP.NET Core**, **Entity Framework Core**, and **JWT Authentication**. Designed to demonstrate core backend engineering principles: RESTful design, database management, data isolation, and clean architecture.

## 🛠️ Tech Stack
- **Framework:** ASP.NET Core Web API
- **Language:** C#
- **Database:** SQL Server + Entity Framework Core
- **Authentication:** JWT (JSON Web Tokens) + BCrypt password hashing
- **Documentation:** Swagger/OpenAPI
- **Architecture:** Clean Architecture, DTOs, Global Exception Middleware, Service Layer

##  Key Features
- ✅ **Full REST API:** CRUD operations with proper HTTP verbs and status codes
- 🔐 **Secure Authentication:** JWT token generation & validation
- 👤 **User Data Isolation:** Users can only create, view, and edit their own tasks
- 📦 **DTO Pattern:** Separation of domain models and API contracts
- 🛡️ **Global Error Handling:** Centralized middleware for clean error responses
- 📖 **Interactive Docs:** Fully documented endpoints via Swagger UI
